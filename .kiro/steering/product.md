# English Meeting Helper - Product Overview

## What It Does
Real-time AI assistant for English meetings. Helps non-native speakers participate effectively through live transcription, translation, and AI-powered suggestions.

## Core Features
- **Live Transcription**: English speech-to-text with <2s latency (AWS Transcribe / OpenAI Whisper)
- **Real-time Translation**: English → Korean translation with <3s latency
- **Quick Translate**: Instant Korean → English text translation for composing responses
- **AI Suggestions**: Context-aware question/response suggestions based on meeting content

## User Interface
- Live/History tabs for transcript viewing with partial/final states
- 2-sentence caption-style translation display
- Quick translate input panel at bottom
- Suggestions panel with customizable prompts
- Mic settings and transcription controls in top bar

## Target Users
Korean speakers participating in English meetings who need real-time language assistance.
